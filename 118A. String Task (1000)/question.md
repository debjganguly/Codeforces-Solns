Petya started attending programming lessons. On his first lesson, he was asked to write a program that processes a given string consisting of uppercase and lowercase Latin letters in the following way:

1. Delete all vowels.  
2. Insert a character `.` before each consonant.  
3. Replace all uppercase consonants with corresponding lowercase ones.  

Vowels are: **A, O, Y, E, U, I** (both uppercase and lowercase).  
All other letters are considered consonants.  

The program takes exactly **one string** as input and should return a **single string** after processing.
